# ZeroJSON

ZeroJSON implements a simplified, UTF-8 JSON reader/writer in C++.

# Usage

Include all of the cpp, h and hpp files in your project.

main.cpp:
```
#include "ZJSON.hpp"

int main() {
 InitZeroTypesLibrary();
 JSON json;
 json.UnitTests();
}
```

# C++ Coverage

Works with C++98 C++11 to C++17 and possibly 19, 20, 21, etc
